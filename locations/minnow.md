# Minnow

> A forest where living trees permanently scent anything they touch.

| | |
|---|---|
| **Type** | Node City |
| **Node Resource** | Living Scent Wood |
| **Node Rating** | Moderate (+30 to +100 amps) |
| **Status** | Integrated |

## Overview

Minnow is a forest city where, as long as a tree is still alive, anything that touches it permanently gains the scent of that wood. The effect is irreversible and lasts forever. Minnow is the source of luxury goods, preservation materials, and ceremonial items across the empire.
