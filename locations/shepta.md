# Shepta

> Instant concrete — pour it, shape it, done.

| | |
|---|---|
| **Type** | Node City |
| **Node Resource** | Whetstone |
| **Node Rating** | Moderate (+30 to +100 amps) |

## Overview

Shepta produces Whetstone — liquid stone that hardens within seconds of being separated from its source pool. Essentially instant concrete. Pour it, shape it, and it sets almost immediately. Shepta is a major construction resource supplier for the empire.
