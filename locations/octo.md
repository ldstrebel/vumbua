# Octo

> The empire's communications hub — home of the Speaking Stones.

| | |
|---|---|
| **Type** | Major Node City |
| **Node Resource** | Speaking Stones |
| **Node Rating** | Major (+100 to +800 amps) |
| **Status** | Core Harmony territory |

## Overview

Octo produces Speaking Stones — one-way communication crystals. The first stone transmits; the second reproduces the message. This is how long-distance communication works across Harmony — essentially magical radio. Speaking Stone networks span the empire, making Octo a critical infrastructure node.
